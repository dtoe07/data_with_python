# Oregon Income Tax Data Analysis & Data Cleaning Exercises

## Description

This project contains two main components:

### 1. Oregon Income Tax Data Analysis (CS410 Final Project)

- Analyzes Oregon state income tax return data by county and tax year.
- Cleans and preprocesses the dataset (removing missing values, standardizing columns, filtering out non-Oregon counties, etc.).
- Performs exploratory data analysis (EDA) to uncover trends in income, tax, and filing behavior across Oregon counties.
- Visualizes distributions, relationships, and trends (e.g., AGI, net tax, return types).
- Answers specific questions about tax patterns, county comparisons, and economic trends.
- Summarizes findings and reflects on data quality and analysis challenges.

### 2. Data Cleaning Exercises

- Contains hands-on exercises for cleaning and preprocessing real-world datasets (e.g., NYC parking violations).
- Demonstrates techniques for handling missing data, removing invalid entries, and analyzing the impact of data cleaning.
- Includes hypothetical scenarios to illustrate the importance of data quality in analysis.

## Files

- `CS410 Final Project.ipynb`: Main notebook for Oregon income tax data analysis.
- `cleaning_data.ipynb`: Data cleaning exercises and tutorials.
- `Historical_personal_income_tax_return_statistics_by_county_and_tax_year_20250604.csv` and `oregon_income_tax.csv`: Datasets used for analysis.
