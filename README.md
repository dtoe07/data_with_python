# Data With Python – Class Assignments & Final Project

This repository contains a sequence of assignments and a comprehensive final project for the CS410 Data With Python course. Each assignment builds foundational data science skills, culminating in an in-depth data analysis project.

## Final Project: Oregon Income Tax Data Analysis

**File:** `CS410 Final Project.ipynb`

For the final project, each student selects a dataset of their own choice to analyze. I chose the Oregon income tax dataset because it is practical, relevant to real-world policy, and likely to contain underlying data quality issues or interesting trends to explore and clean. The project includes:

- Data cleaning and preprocessing (handling missing values, standardizing columns, filtering non-Oregon counties, etc.)
- Exploratory data analysis (EDA) to uncover trends in income, tax, and filing behavior
- Visualizations of distributions, relationships, and trends (e.g., AGI, net tax, return types)
- Answers to specific questions about tax patterns, county comparisons, and economic trends
- Summary of findings, limitations, and reflections on the analysis process

## Assignments Overview

Each assignment notebook focuses on a core data science concept or tool. Below is a brief summary of each:

- **0_data_structure_workouts (1).ipynb**
  - _Objective:_ Practice Python’s built-in data structures (lists, sets, tuples, dictionaries).
  - _Learning Outcome:_ Gain fluency in manipulating and applying core data containers for later data analysis tasks.

- **1_numpy_workouts.ipynb**
  - _Objective:_ Learn the basics of NumPy arrays and operations.
  - _Learning Outcome:_ Understand array creation, manipulation, and vectorized computation for efficient numerical analysis.

- **2_pandas_series_workouts.ipynb**
  - _Objective:_ Work with Pandas Series, the fundamental one-dimensional labeled array.
  - _Learning Outcome:_ Create, access, manipulate, and analyze data within Series objects.

- **3_pandas_dataframe_workouts.ipynb**
  - _Objective:_ Explore Pandas DataFrames for tabular data analysis.
  - _Learning Outcome:_ Learn to load, inspect, manipulate, and analyze datasets using DataFrames.

- **4_cleaning_data.ipynb**
  - _Objective:_ Practice data cleaning techniques on real-world datasets.
  - _Learning Outcome:_ Identify and handle missing or invalid data, and understand the impact of data quality on analysis.

- **5_intro_visualizations.ipynb**
  - _Objective:_ Build basic data visualizations using matplotlib and pandas.
  - _Learning Outcome:_ Create and interpret plots to explore and communicate data insights.

- **6_aggregating_data.ipynb**
  - _Objective:_ Aggregate and summarize data using groupby and related operations.
  - _Learning Outcome:_ Perform grouping, summarization, and basic analysis on larger datasets.
